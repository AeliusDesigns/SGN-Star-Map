SGN PASSWORD HASHER

Generate a hashed config file for the SGN editor lock. Run locally, never online.

Steps:

1. Create a file called .sgn_cfg in your project root (same folder as index.html).

2. Paste the JSON above into that file.

3. Make sure .sgn_cfg is in your .gitignore.

Do NOT commit this file. Do NOT share it. The hash is one-way but the salt makes brute-forcing significantly harder.
